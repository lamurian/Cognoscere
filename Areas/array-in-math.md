## Definition

An **array** in mathematics is an orderly arrangement of numbers, symbols, or expressions organised in **rows and columns**. It is one of the most fundamental structures in mathematics, forming the building block for vectors, matrices, and tensors.

## Key characteristics

- **Rectangular arrangement** — elements are laid out in a grid-like structure.
- **Ordered** — each element has a specific position, identified by its row and column indices.
- **Homogeneous** — typically all elements are of the same type (e.g., all real numbers).

## Dimensionality

| Dimension | Name | Description |
|-----------|------|-------------|
| 1D | Vector | A single row or column of elements. |
| 2D | Matrix | A grid of elements with rows and columns. |
| 3D+ | Tensor | A multi-dimensional array extending beyond two dimensions. |

- A **1-dimensional array** (vector) has a single index per element.
- A **2-dimensional array** (matrix) has two indices (row, column) per element.
- A **3-dimensional or higher array** (tensor) has three or more indices.

## Notation

A 2×3 array (2 rows, 3 columns) is typically written as:

```
A = [ a₁₁  a₁₂  a₁₃ ]
    [ a₂₁  a₂₂  a₂₃ ]
```

The element at row *i*, column *j* is denoted *aᵢⱼ*.

## Common operations

- **Indexing** — accessing an element by its position (e.g., *aᵢⱼ*).
- **Transposition** — flipping rows and columns.
- **Arithmetic** — addition, subtraction, and multiplication (with certain constraints).
- **Reshaping** — changing the dimensions without altering the data.

## Applications

- **Linear algebra** — vectors and matrices are arrays.
- **Statistics** — data tables, contingency tables.
- **Computer science** — arrays are the core data structure in programming.
- **Physics** — tensors represent physical quantities in multiple dimensions.
- **Machine learning** — feature matrices, weight matrices, and activation arrays.
