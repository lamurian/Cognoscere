## Summary

The Martin-Key et al. (2022) systematic review identified substantial methodological limitations across 28 digital mental health assessment validation studies, many using MINI as reference standard. These limitations constrain confidence in reported diagnostic accuracy.

## Key Points

### Major Risk of Bias Domains (QUADAS-2)

| Domain | High Risk | Unclear Risk |
|---|---|---|
| Patient Selection | 43% (12/28) | 32% (9/28) |
| Index Test | 36% (10/28) | 46% (13/28) |
| Reference Standard | 4% (1/28) | 50% (14/28) |
| Flow and Timing | 14% (4/28) | 39% (11/28) |

### Specific Limitations

**Patient Selection**
- Case-control designs common (enriching with known cases)
- Convenience sampling from clinics or online panels
- Exclusion of comorbid/complex presentations
- University student samples not representative

**Index Test Administration**
- Few studies prespecified diagnostic thresholds
- Adaptive tools: algorithms often proprietary/not transparent
- Unclear if index test interpreted blind to reference standard
- Variable completion settings (supervised vs unsupervised)

**Reference Standard (MINI/SCID/CIDI)**
- 50% unclear if reference standard interpreted blind to index test
- MINI administered by telephone vs in-person — mode effects unreported
- Variable interviewer training and fidelity monitoring
- Some studies used abbreviated MINI modules only

**Flow and Timing**
- Interval between index and reference: same day to 34 days
- 14% high risk: reference not re-administered if completed recently
- Nonpsychiatric controls sometimes not given reference standard
- Missing data handling rarely reported

### Applicability Concerns
- Most studies in primary care/general population (64%)
- Limited secondary care validation (43%)
- Few studies in diverse ethnic/cultural groups
- Digital literacy/access bias in web-based tools
- Commercial tools (CAT-MH, CMFC) — independent replication needed

### Recommendations for Future Studies
1. Use consecutive/recruited sampling in intended use setting
2. Prespecify thresholds; register analysis plans
3. Ensure blinding of reference standard assessors
4. Report interval between tests; justify clinically
5. Include diverse populations; report subgroup analyses
6. Follow STARD guidelines for diagnostic accuracy reporting
