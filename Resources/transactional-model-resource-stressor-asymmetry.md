Obbarius et al. [@obbarius2021] validated the core of Lazarus and Folkman's Transactional Model using SEM in a psychosomatic sample (n=2,216). Personal resources (sense of coherence, self-efficacy, optimism) had a far stronger effect on stress response (beta = -0.73, p < 0.001) than perceived external stressors (beta = 0.34, p < 0.001), together explaining 91% of variance. Resources also negatively influenced perceived stressors (beta = -0.51), meaning resource-rich individuals perceive fewer demands.

The resource effect (beta = -0.73) is roughly double the stressor effect (beta = 0.34). This validates asymmetrical weighting in computational threshold models like eta_eff = eta_0 + eta_chi * chi - eta_zeta * zeta. However, the simulation gives hindrance (0.25) larger short-term weight than challenge (0.15), reflecting negativity bias. The empirical finding suggests long-term resource buffering should dominate, which the resilience dynamics already capture through homeostatic return and protective factor boosts. Coping could not be modeled as an independent mediator, so coping success in simulation is best treated as an emergent outcome rather than a separate pathway.
