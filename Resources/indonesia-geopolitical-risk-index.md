The Indonesia Geopolitical Risk (GPR) Index is an open-source Python pipeline at https://github.com/billycemerson/indo-gpr-index that scrapes five Indonesian news sources daily: Antara, Detik, Kompas, Tempo, and Tribunnews. It calculates a GPR index across five dimensions (war threat, peace threat, military buildup, war acts, terror acts) using the Caldara and Iacoviello methodology. The pipeline runs daily at 08:00 WIB via GitHub Actions and exports results to Google Sheets. This is the only publicly accessible daily geopolitical risk measure specifically for Indonesia.
