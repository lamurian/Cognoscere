Engineered plant root materials (EPRMs) use obstacles to direct root growth into planar grid structures with predictable mechanical behavior [@melo2022]. This creates root-soil composites that function like geogrid reinforcement in MSE walls but are biological, self-repairing, and strengthen as plants mature.

For living fences, the foundation system can match concrete footing performance while being entirely biological. Species selection controls root architecture: deep taproots (Leucaena leucocephala to 3+ meters, Lannea coromandelica) provide deep anchorage, while fibrous roots (willow, poplar) create dense near-surface mats preventing undermining.

Combining both root types in a multi-species living fence creates foundation redundancy. The root grid resists lateral loads, prevents erosion, and self-repairs damage -- functions that concrete footings cannot perform without replacement.

## Key Points
- Obstacle-directed root growth produces predictable grid reinforcement.
- Root-soil composites = biological geogrid, self-repairing, strengthening.
- Deep taproots for anchorage + fibrous roots for undermining prevention.
- Multi-species root systems create foundation redundancy.
