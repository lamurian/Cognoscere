## Summary

The Challenge-Hindrance Stressor Framework (CHSF) distinguishes between challenge stressors (demands offering growth potential) and hindrance stressors (demands impeding growth) [@cavanaugh2000]. This distinction is foundational to the appraisal-based simulation of stress responses.

The appraisal function z = omega_C * c - omega_O * o + b transforms event attributes (controllability c, overload o) into a continuous appraisal score. The sigmoid transformation chi = sigma(gamma * z) converts this into challenge probability [@podsakoff2023]. Higher gamma values (>6) produce more decisive challenge/hindrance classification, while lower values create graded appraisal.

Meta-analytic evidence [@bakker2017] confirms challenge stressors positively predict engagement and performance (rho=0.21-0.35), while hindrance stressors negatively predict these outcomes (rho=-0.18 to -0.32). The CHSF 2.0 framework emphasizes that appraisal is dynamic and context-dependent, with omega weights varying by individual and situation.
